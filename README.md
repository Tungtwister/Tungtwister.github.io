## 📦 Installation
To run the website locally, follow these steps:

1. Clone this repository:
   ```bash
   git clone https://github.com/Tungtwister/Tungtwister.github.io.git
2. Install dependencies:
   ```bash
   npm install
3. Start the development server:
   ```bash
   npm start
